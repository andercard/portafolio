Hola mi nombre es Andersson Mesa. Nací hace 24 años en Cúcuta – Colombia. Soy Tecnólogo en desarrollo de sistemas informáticos apasionado por el desarrollo web con más de 3 años de experiencia en el área frontend.

Actualmente estoy terminando la ingeniería de sistemas y trabajando en proyectos remotos como frontend. La tecnología nunca se detiene y a mí me encanta siempre estar actualizado en las herramientas que trabajo.

Conocimientos

En el tiempo de experiencia que tengo desarrollando e creado sitios desde cero con las tecnologías bases del desarrollo web que son html, css y javascript pero tambien transformo los sitio al cms wordpress.

Me encanta el responsive Web Design (mobil first) manejo los framework boostrap y foundation pero no me limito a esas herramientas de columnas de css también trabajo en la maquetacion de sitios sin usar ese tipo de tecnologías, también en el desarrollo de sitios pwa (progressive web app).

Uso librerías de javascript como jquery o framework como vuejs, preprocesadores como jade y sass, gulp como task runner para la compilación,minificación y optimización de html, css y javascript. Llevando siempre como respaldo el control de versiones con git.
